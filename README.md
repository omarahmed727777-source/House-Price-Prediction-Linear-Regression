# House Price Prediction using Linear Regression
This project implements a Simple Linear Regression model to predict house prices based on various features.

## Key Features
- **Data Analysis:** Performed feature correlation and coefficient interpretation.
- **Model Performance:** Achieved an **R-squared value of 0.6688**, explaining approximately 67% of the data variance.
- **Evaluation:** Utilized Mean Squared Error (MSE) and visualization of Actual vs. Predicted prices.

## Technologies Used
- Python, Pandas, Scikit-learn, Matplotlib.
